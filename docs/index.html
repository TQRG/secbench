SECBENCH

#security#fixes#vulnerabilities#benchmark
Dataset of 676 security vulnerabilities fixes. In 2017, we mined the commits messages of 238 projects using regular expressions for each vulnerability (cf. Patterns). In 2020, we classified vulnerabilities using the CWE taxonomy. Some vulnerabilities contain the score and severity information (CVEs). If you use our dataset, please, cite one of the papers below.
